# Apache

When we need to serve webpages or web applications. HTTPD server ~ HTTP Daemon.
When a http request comes it will listen to that request and serve the content.

eg. example.com -> DNS -> IP address -> Apache server (httpd) -> serve the content.

Common use cases:
- Hosting websites
- Reverse proxy
- Load balancing
- Multi site hosting (shared hosting)
- HTTPS/SSL termination
- Access control and authentication
